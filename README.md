# Mirai Discord Bot

Send mirai attacks using a discord bot.

### How to use

* **Create .env file**

* **Edit .env with ur config**

* **Add ur mirai in db.json**

* **to install all dependencies (npm i)**

* **run this bot (npm start)**

### Commands

* **attack**

* **clear**

* **eval**

* **help**

* **methods**
